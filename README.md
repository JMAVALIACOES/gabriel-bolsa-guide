## 🔧 Como Fazer Deploy

### Opção 1: GitHub Pages (Recomendado - Automático)

1. **Crie um repositório no GitHub**:
   ```bash
   git init
   git add .
   git commit -m "Initial commit: Gabriel Bolsa Guide"
   git branch -M main
   git remote add origin https://github.com/SEU_USUARIO/gabriel-bolsa-guide.git
   git push -u origin main
   ```

2. **Ative GitHub Pages**:
   - Vá para Settings → Pages
   - Source: Deploy from a branch
   - Branch: main, folder: / (root)
   - Salve

3. **Seu site estará disponível em**:
   ```
   https://SEU_USUARIO.github.io/gabriel-bolsa-guide/
   ```

### Opção 2: Vercel (Alternativa)

1. Conecte seu repositório GitHub ao Vercel
2. Vercel detectará automaticamente e fará deploy
3. Link: `https://gabriel-bolsa-guide.vercel.app/`

### Opção 3: Netlify (Alternativa)

1. Conecte seu repositório GitHub ao Netlify
2. Netlify fará deploy automático
3. Link: `https://gabriel-bolsa-guide.netlify.app/`
